Step 1:

Select one or more crime types using the dropdown next to "Search Crime Types."

Step 2:

Select a start date and end date using the calendar tool.

Step 3:

Click the "Map Crimes" button.

Step 4:

Click a popup on the map to get more info about that crime. Then, click the "Nearby Tweets" button to see tweets that are geotagged with a location near to the crime location.
