How to use this app:

Step 1:

Select one or more crime types using the multiselect dropdown.

Step 2:

Select a date range using the calendar tool.

Step 3:

Click the "Map Selected Crimes" button.

Step 4:

Click a popup on the map to get more info about that crime. Then, click the "Nearby tweets" button to see tweets that are geotagged with a location near to the crime location.
